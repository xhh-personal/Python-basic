## 1. 安装过程

## 2. 一些设置

* 设置编辑器字体 File --> Settings --> Editor --> Font
* 设置控制台字体 File --> Settings --> Editor --> Font --> Color Scheme --> Console Font
* 选择Python解释器 File --> Settings --> Project:xx --> Python Interpreter
